THE MANI BAR BY SAVI - PORTFOLIO WEBSITE

What is inside:
- index.html  -> the website
- styles.css  -> all design and mobile styling
- script.js   -> socials, WhatsApp booking, QR code and page switching
- assets/     -> images, logo and QR code

How to use:
1. Open index.html in your browser to preview.
2. Upload everything in this folder to GitHub Pages, Netlify, Vercel, or your hosting.
3. Open script.js and update these values when Savina gives them to you:
   - instagram
   - tiktok
   - phone
   - whatsappNumber
   - location
   - siteUrl

QR code:
- The site automatically tries to generate a QR code for the live website link when hosted online.
- If you already know the final website link, paste it inside script.js as siteUrl.
- The backup QR image is assets/qr-portfolio.png.

Recommended final info to collect from Savina:
- Instagram handle
- TikTok handle
- WhatsApp booking number
- Phone number
- Exact location or area
- Price list
- Working days/hours
- Any booking rules, deposit policy, or home service info
